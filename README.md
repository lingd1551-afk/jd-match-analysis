# AI Career Coach - JD匹配度分析

一个AI驱动的职业教练平台，提供JD（岗位描述）匹配度分析功能。

## 功能特性

- 📤 **JD上传**：支持粘贴或输入完整的岗位描述
- 🔍 **智能分析**：自动提取JD中的关键信息（岗位名称、技能要求、核心要求等）
- 📊 **匹配度评分**：基于简历和JD的匹配度计算综合得分
- ✅ **优势分析**：识别简历与岗位匹配的优势点
- ⚠️ **待提升项**：指出需要改进的地方
- 💡 **优化建议**：提供针对性的简历优化建议
- 📝 **简历修改建议**：给出具体的简历修改方向

## 技术栈

- **前端框架**：Next.js 14 (App Router)
- **UI框架**：React 18
- **样式**：Tailwind CSS
- **语言**：TypeScript

## 快速开始

### 安装依赖

```bash
npm install
```

### 运行开发服务器

```bash
npm run dev
```

打开 [http://localhost:3000](http://localhost:3000) 查看应用。

### 使用流程

1. 访问 `/jd-match` 页面
2. 在文本框中粘贴或输入完整的岗位描述（JD）
3. 点击"开始分析"按钮
4. 系统会自动分析并生成匹配度报告
5. 查看详细的匹配报告，包括：
   - 总体匹配度评分
   - 优势分析
   - 待提升项
   - 优化建议
   - 简历修改建议

## 核心功能说明

### JD分析算法

系统会从JD文本中提取：
- 岗位名称
- 技能关键词（React、Vue、TypeScript等）
- 核心要求

### 匹配度计算

匹配度基于以下因素：
- 技能匹配度（60%权重）
- 经验匹配度（40%权重）

### 报告生成

系统会生成包含以下内容的详细报告：
- 匹配度评分（0-100分）
- 优势分析列表
- 待提升项列表
- 优化建议（5-7条）
- 简历修改建议（5条）

## 注意事项

- 当前版本使用模拟简历数据进行匹配分析
- 实际应用中，简历数据应从数据库或用户输入获取
- 可以集成OpenAI API等AI服务来增强分析能力
